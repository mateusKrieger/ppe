    • O que é a BIOS e qual sua função ?

    BIOS, que significa Basic Input/Output System (Sistema Básico de Entrada/Saída), é um firmware embutido na placa-mãe dos computadores. É responsável por inicializar e testar o hardware do sistema durante o processo de boot (inicialização) e fornecer serviços de runtime para os sistemas operacionais e programas.


    Aqui estão algumas das funções principais do BIOS: Inicialização do Hardware: Quando você liga o computador, o BIOS executa uma série de testes chamados POST (Power-On Self Test) para garantir que o hardware está funcionando corretamente. Carregamento do Sistema Operacional: O BIOS localiza e carrega o sistema operacional na memória RAM, a partir de um dispositivo de armazenamento, como um disco rígido, SSD, CD/DVD ou uma unidade USB. Configuração do Hardware: O BIOS permite que os usuários ajustem configurações de hardware através de uma interface de configuração (geralmente acessada pressionando uma tecla específica durante a inicialização, como Del, F2 ou F10). Gerenciamento de Periféricos: O BIOS facilita a comunicação entre o sistema operacional e os dispositivos de hardware, como teclados, mouses, discos rígidos e outras unidades de armazenamento.
